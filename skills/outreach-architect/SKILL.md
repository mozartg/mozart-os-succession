# Outreach Architect

Trigger: any outreach batch (Upwork proposals, cold DM/email, gig replies).

## Method
1. One angle per batch: lead with the prospect's specific observed flaw (from the audit), never a generic capability pitch.
2. Proof asset attached: every message carries a leave-behind (1-page finding, Loom, PDF snippet) that has passed the lie-detector.
3. Positioning line: "I don't sell AI output. I sell verified, ready-to-ship deliverables — every claim proven before it reaches you."
4. Sequence: initial + 2 follow-ups (day 3 value-add, day 7 close-the-loop). Draft all three at once.
5. Fixed-scope pricing in the first message ($150-400 range for audits/installs). No "let's hop on a call" as the only CTA.
6. Human sends. Drafts only — Mo hits send.

## Quality checklist
- [ ] Each message cites a real, verified flaw of that prospect
- [ ] Proof asset attached and lie-detector-passed
- [ ] All 3 sequence steps drafted
- [ ] Price and scope stated

## Rerunnable prompt
"Act as outreach-architect. Prospects: [list with audited flaws]. Draft the 3-step sequence per prospect with the proof asset reference, fixed price, and the verified-deliverables positioning line."
